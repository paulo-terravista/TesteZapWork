Execução do Micro Serviço:

	É necessário ter o java 1.8 instalado.
	Para executar a aplicação execute na linha de comando do DOS (Cmd)
	
		java -jar LimiteCreditoService.jar
	
	Aguarde até a inicialização acabar.
	
Execução da Aplicação web
	
	É necessário ter instalado o npm (de preferencia a versão mais atual) e o angular 5 na máquina.
	
	Para executar vá até a pasta da aplicacao limite-credito e execute:
	
		npm install
	
	O npm fará o download de todas as bibliotecas de dependências, após o término execute na linha de comando:
	
		ng serve
		
	Aguarde a aplicação subir.
	
	
No browser informa a URL http://localhost:4200
